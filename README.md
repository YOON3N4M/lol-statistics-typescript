
# lol-statistics-typescript


### 소개

라이엇 게임즈에서 제공하는 API를 활용해 리그오브레전드 플레이어의 정보를 조회 할 수 있습니다. <br/>
최근 20게임의 전적, 모스트 7, 포지션 선호도, 승률과 kda 등의 통계 정보 등을 제공 합니다.<br/>
전체적인 디자인과 구성은 [OP.GG](https://www.op.gg/)를 참고하여 제작 되었습니다.
